Aplicación creada mediante Visual Studio en C#, usando la plataforma de .NET Framework 4.8 para su implementación.

Este programa sirve para realizar el cotejo de información de fichas catalográficas existentes en Prometeo V, cotejado con una tabla de Excel para enviar las comprobaciones y correcciones de errores a un servicio web de Google Forms para su corrección en el área laboral correspondiente.

Por ende, este programa podrá solo ser usado sólo en el equipo que fué instalado, no es posible portarlo a otro equipo.
Si así se hiciese, este programa se bloqueará automáticamente.
